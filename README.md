# Element Clone Generator

Интерактивное приложение на C# Windows Forms, которое создает копии элементов управления при клике на исходные.

## Функциональность

- **Исходные элементы** (создаются при запуске):
  - Кнопка → при клике создает новые кнопки
  - Метка → при клике создает новые метки
  - Поле ввода → при клике создает новые текстовые поля

- **Создаваемые элементы**:
  - Автоматическая нумерация ("Кнопка 1", "Метка 2", "Поле 3")
  - Случайное расположение в пределах формы
  - Сохранение размеров исходных элементов

- **Счетчик**:
  - Отображение количества созданных элементов каждого типа

## Интерфейс

- Исходные элементы в левом верхнем углу:
  - Кнопка (100×30)
  - Метка ("Метка")
  - Поле ввода (150×20)
- Счетчик под исходными элементами

## Пример работы

1. Клик по исходной кнопке → появляется "Кнопка 1" в случайном месте
2. Клик по исходной метке → появляется "Метка 1" в случайном месте
3. Клик по исходному полю → появляется "Поле 1" в случайном месте
4. Счетчик показывает: "Кнопок: 1  Меток: 1  Полей: 1"

- В примеры обработки событий

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

При каждом клике создается новый элемент того же типа, что и исходный. Координаты новых элементов генерируются случайным образом в пределах видимой области формы.
